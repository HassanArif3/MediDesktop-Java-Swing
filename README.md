# MediDesktop: Java-Swing Hospital Management System

[cite_start]**MediDesktop** is a professional-grade software solution designed to optimize organizational and medical processes in healthcare facilities[cite: 80, 81, 266]. [cite_start]Developed as a term project at the **University of Gujrat**, this system replaces inefficient paper-based records with a secure digital network [cite: 13, 277-279].

## 📊 Project Metadata
* [cite_start]**Version:** 1.0 (Approved) [cite: 10]
* [cite_start]**Development Date:** April 2025 [cite: 14]
* [cite_start]**Developers:** M. Hassan Arif, M. Haseeb Hafeez, and Faran Mir [cite: 11, 274]
* **Documentation:** [Software Requirements Specification](docs/MediDesktop-SRS-v1.0.pdf) & [Technical Design Report](docs/MediDesktop-Technical-Project-Report.pdf)

## 🛠 Technical Stack
* [cite_start]**Language:** Java SE [cite: 121]
* [cite_start]**GUI Framework:** Java Swing / JFrame (Developed in NetBeans) [cite: 121, 530]
* **Database:** SQL Server 2022 / MySQL with JDBC connectivity
* [cite_start]**API Integration:** Payment gateways (Easypaisa/Jazzcash) and SMS notification systems [cite: 171, 173]

## 🚀 Key Modules & Functional Requirements
The system is built around four core functional pillars:
* [cite_start]**Registration:** Allows users to securely sign up and log in using email and password validation [cite: 58, 84-86, 178-180].
* [cite_start]**Doctor Management:** Enables administrators to add, modify, view, and delete doctor profiles [cite: 66, 87-91, 183-186].
* [cite_start]**Appointment Management:** A digital process for booking and canceling appointments, featuring a token system for patient identification [cite: 59, 60, 92-96, 189-192, 439].
* [cite_start]**Patient Records:** Empowers doctors to add, update, and view medical history via a unique patient ID [cite: 61-63, 100, 150, 195-197].
